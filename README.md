## ğŸ¨ Customization

### Changing Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #198754;    /* Green */
    --secondary-color: #0d6efd;  /* Blue */
    --accent-color: #ffc107;     /* Yellow */
}
```

### Updating Content
- Edit HTML files directly for text content
- Replace image URLs with your own images
- Update contact information in `contact.html`

### Changing Google Maps Location
1. Go to Google Maps
2. Search for your location
3. Click "Share" â†’ "Embed a map"
4. Copy the iframe src URL
5. Replace in `contact.html`

2. Update the content
3. Add link to navigation in all pages
